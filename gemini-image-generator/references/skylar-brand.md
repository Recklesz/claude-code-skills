## Data Visualization DNA

### The Core "DNA" Template

Every data visualization prompt **must** include these elements:

**1. The Setup (Start of Prompt):**
```
High-fidelity, professional SaaS dashboard cards sitting on a premium liquid wave background...
```

**2. The Background Rule:**
```
...A high-resolution, detailed abstract background of flowing silk waves in Skylar Cyan (#8EE6F7) and Deep Teal (#37BCD9). It looks like a premium wallpaper.
```

**3. The UI Container Rule:**
```
...Clean matte white cards with a very thin, subtle light-grey border and soft teal-tinted drop shadows. They look like crisp native app components.
```

**4. The Typography Rule:**
```
...Text is clear and legible in Ubuntu font (rounded sans-serif). Main Headings are Dark Navy (#0E2E3E). Labels are Grey.
```

### Full Prompt Template

Fill in the [Bracketed Sections]:

```
[Number of Cards] high-fidelity, professional SaaS dashboard cards sitting on a premium liquid wave background.

The Background: A high-resolution, detailed abstract background of flowing silk waves in Skylar Cyan (#8EE6F7) and Deep Teal (#37BCD9).

The Cards: Clean matte white cards with a very thin, subtle light-grey border and soft teal-tinted drop shadows.

Typography: Text is clear and legible in Ubuntu font. Main Headings are Dark Navy (#0E2E3E).

[Card 1 Detail]: [Describe layout: e.g., A vertical bar chart with one Teal bar and one Grey bar. Label reads "X".]
[Card 2 Detail]: [Describe layout: e.g., A crisp radial progress ring in Teal. Text inside reads "Y%".]

Style: "Clean UI", pixel-perfect alignment, flat design with soft depth, white backgrounds, professional application interface, no neon effects.
```

## Component Recipes

### For Comparisons (e.g., 8.4 vs 9)
```
Two vertical bars side-by-side. The main bar is solid Vibrant Teal (#37BCD9). The target bar is outlined in grey or lighter teal.
```

### For Percentages (e.g., 39%)
```
A crisp, flat-style circular progress ring in Vibrant Teal. No glow effects. Text placed clearly in the center.
```

### For Increases (e.g., +17%)
```
A small, crisp Success Green (#2A7D5F) upward arrow icon placed next to the percentage.
```

### For Time/Volume (e.g., Minutes)
```
A minimalist, flat-style 3D icon of [clock/hourglass] in soft Teal. Non-intrusive.
```

### Information Boxes
```
Background: rgba(55, 188, 217, 0.15)
Border left: 4px solid #37bcd9
Padding: 18px 20px
Border radius: 6px
Box shadow: 0 2px 5px rgba(14, 46, 62, 0.08)
```

### Stats Cards
```
Background: rgba(55, 188, 217, 0.08)
Border: 2px solid rgba(55, 188, 217, 0.2)
Border radius: 10px
Padding: 20px
Large numbers: 32px, bold, primary color
Labels: 14px, gray (#718096)
```

| Feature | **DO SAY (Skylar Look)** | **DON'T SAY (Generic Look)** |
|:---|:---|:---|

| **Colors** | "Dark Navy (#0E2E3E)," "Vibrant Teal," "Skylar Cyan (#8EE6F7)" | "Black," "Blue," "Neon Blue" |
| **Material** | "Clean White," "Paper-like," "Soft Depth" | "Glass," "Transparent," "Glowing" |
| **Charts** | "Crisp," "Flat-style," "Solid Fill" | "3D Exploded," "Neon Glow," "Sketchy" |
| **Status** | "Success Green (#2A7D5F)" | "Bright Green," "Lime" |
| **Background** | "Liquid waves," "Flowing silk," "Premium wallpaper" | "Abstract," "Geometric," "Pattern" |
| **UI Elements** | "Matte white cards," "Native app components" | "Glass cards," "Frosted glass" |

## Design Principles

1. **Functional Beauty** - Clean, purposeful design that enhances usability
2. **Matte over Glossy** - Soft, professional finishes rather than shiny, glass effects
3. **Flat with Soft Depth** - Minimal shadows, subtle elevation, no harsh 3D effects
4. **Teal as Hero** - Primary blue/teal colors dominate, with Success Green for positive metrics
5. **Clean Typography** - Ubuntu font, clear hierarchy, legible at all sizes
6. **Liquid Backgrounds** - Flowing wave patterns in brand colors for visual interest
7. **White Containers** - Crisp white cards/containers against colored backgrounds

## Pro Tips for Image Generation

### For Complex Text
If the AI misspells complex text:
1. Simplify the prompt to generate layout and numbers only
2. Add text overlays using external tools for perfect spelling

### For Brand Consistency
- Always use exact hex codes in prompts (#37bcd9, #8ee6f7, #0e2e3e)
- Mention "Ubuntu font" explicitly
- Use vocabulary from the cheat sheet
- Reference "liquid waves" or "flowing silk waves" for backgrounds
- Specify "matte white cards" for containers
- Include "no neon effects" to avoid generic looks
